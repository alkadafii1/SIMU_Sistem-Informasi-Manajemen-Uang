# SIMU - Sistem Informasi Manajemen Uang

**Aplikasi pencatat dan perencana keuangan pribadi untuk generasi muda**

SIMU membantu pengguna mencatat pemasukan & pengeluaran, menyusun anggaran otomatis, memberikan rekomendasi pengeluaran harian, serta menganalisis pola keuangan menggunakan deep learning clustering.

---

## Fitur

| Modul | Fitur |
|-------|-------|
| Onboarding | Input profil singkat (pendapatan, tujuan keuangan opsional) |
| Pencatatan | CRUD pemasukan & pengeluaran, riwayat transaksi |
| Smart Budgeting | Alokasi otomatis 50-30-20, rekomendasi harian, notifikasi sisa anggaran |
| AI Analytics | Clustering pengguna & prediksi kesehatan finansial |
| Dashboard | Ringkasan keuangan, grafik pengeluaran, insight bulanan |
| Rollover | Data keuangan otomatis dibawa ke bulan berikutnya |
